# AI University Summit — Interactive Session Guide

**Find the right sessions for you.**

An interactive guide that helps summit attendees navigate the day's schedule based on their role. Select whether you're a classified professional, faculty member, administrator, employer/community partner, or new to AI, and the guide highlights the sessions most relevant to you.

### [Open the Guide](https://ljfvalmonte.github.io/ai-university-summit-guide/)

---

## What This Tool Does

The AI University Summit brings together community college faculty, K-12 educators, administrators, classified professionals, and community partners for a day of AI-focused professional development. This interactive guide helps attendees plan their day.

### Features

- **Role-Based Filtering** — Select "I am a . . ." to see sessions tailored to your role (Classified Professional / Staff, Faculty / Teacher, Administrator, Employer / Community Partner, New to AI)
- **Day-at-a-Glance Timeline** — Full schedule with time blocks, session titles, and track labels (101 Track, etc.)
- **Session Details** — Expand any session to see descriptions, presenters, and room locations
- **Mobile-Friendly** — Works on phones so attendees can use it on-site during the event

## For Future Summits: How to Update

The session schedule, presenter names, track labels, and role filters can all be updated by editing the source files in this repo. The basic workflow:

1. Edit the session data (times, titles, descriptions, track assignments, role tags)
2. Commit changes to the `main` branch
3. GitHub Pages automatically redeploys the updated guide

If you need help adapting the guide for a new event, bring the updated schedule into a Claude conversation and request the code changes.
